数据库关系表暂设置为与课程选修有关的模式：
1.基本信息表：学号 姓名 性别 所属学院  
2.本学期成绩表：学号，姓名，科目，成绩，
3.科目信息表：科目代码，科目简介，该科学分，授课老师
当然为了信息的隔离，在查询时，学生只能查询自身相关信息，其他学生信息不予展示
管理员账号可以查询任何信息，并可进行增删改查操作，学生仅可进行查询操作

  管理员可对任意总表进行增删改查。
